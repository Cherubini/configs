# Proxy Linux Embraer #
> Compatible with Ubuntu 16.04 LTS or later

## Setup

*Setup configure credentials file and create proxy link into /usr/local/bin.*

```
./setup.sh <embraer-username> <embraer-password>
```

### Proxy set
- Gnome (Unity base)
- Docker
- APT
- Environment
- Git
- Node
- Bower


## Commands
```
proxy start
proxy stop
```
